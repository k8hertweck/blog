I wrote my first blog post in July, then promptly neglected any subsequent posts until December. Classic me!

Shortly after writing that first post, I decided to switch up how the site was built. My initial instinct had been to use a Jekyll site with the Minimal Mistakes theme, an approach that was familiar to me from websites I support during my day job at [Fred Hutch]({{< ref “/projects” >}}).

This website is built using the Minimal Mistakes Jekyll theme and is deployed using a public GitHub repository and GitHub Pages.
